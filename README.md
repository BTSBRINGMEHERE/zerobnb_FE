## 🙌 여행, 숙박 예약 서비스 웹 애플리케이션

## 프로젝트 개요
airbnb, 야놀자 등등 여행,숙박 예약 사이트의 기본,핵심 기능을 참고해 웹 애플리케이션 서비스로 개발을 진행하였습니다.

### 프로젝트 목표 
숙박 예약 사이트의 기본, 핵심 기능 서비스를 제공한다. 

## 🛠 기술 스택
- 프론트엔드 
  - React.js, typescript
- 백엔드
  - Spring Boot, JPA, mybatis, QueryDsl
- DB
  - MySql, Redis
- infra
  - AWS, nexus, sonarqube, docker
